RStudio Shortcuts

Boost your efficiency with these handy RStudio shortcuts:

  • Cmd + Shift + P: Open command palette
  • Cmd + Shift + C: Comment/uncomment code
  • Cmd + C: Copy text
  • Cmd + L: Clear the console
  • Cmd + S: Save file
  • Cmd + Enter: Run selected code
